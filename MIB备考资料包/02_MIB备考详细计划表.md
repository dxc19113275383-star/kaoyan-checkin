# 国际商务专硕（MIB）备考详细计划表

> **目标院校**：上海对外经贸大学（主攻）／浙江工商大学（保底）
> **考试科目**：101政治 + 204英语二 + 396经济类综合（数学70+逻辑40+写作40）+ 434国际商务专业基础（上经贸含50分专业英语）
> **目标初试**：2027年12月下旬（2028级）
> **背景**：音乐表演跨考、英数较弱
> **时间线**：2026全年在职（每天2-3h）→ 2027年初辞职全职（每天8-10h）

---

## 总体战略（一句话）

> **在职这一年，只干两件事：英语打到能读、数学补到能算。** 逻辑、写作、434、政治全部留到全职阶段。在职期贪多必崩，守住"英语+数学基础"这条线，全职期就能顺势起飞。

---

## 阶段一：在职打基础期（现在 → 2026年12月，约8个月）

### 阶段总目标
- 英语二：单词背完3轮、语法补全、能独立拆解长难句、阅读真题精读起步
- 396数学：从零补到"看得懂例题、做得出基础题"，覆盖微积分+线代+概率基础
- 其余科目：不碰（逻辑/写作/434/政治留给全职阶段）

### 每日时间分配（2-3h，雷打不动）
| 时段 | 时长 | 内容 |
|------|------|------|
| 通勤/晨起 | 30min | 英语单词APP打卡 |
| 晚间第一段 | 60min | 396数学（视频课+跟做例题） |
| 晚间第二段 | 45min | 英语语法/长难句 或 阅读精读 |
| 睡前 | 15min | 单词回顾+数学公式默写 |
> 累的那天保底只做30min单词，**绝不零打卡**。

### 月度目标拆解

| 月份 | 英语二月目标 | 396数学月目标 |
|------|-------------|--------------|
| 第1月 | 单词第1轮启动（核心高频词）；语法框架视频课过1遍 | 补初高中：函数、不等式、基础代数运算 |
| 第2月 | 单词第1轮完成；语法重点章节（时态、从句、非谓语） | 极限与连续、导数概念入门 |
| 第3月 | 单词第2轮启动；长难句拆解专项训练 | 导数应用、不定积分基础 |
| 第4月 | 单词第2轮完成；开始阅读真题精读（每周2篇） | 定积分、多元函数微分基础 |
| 第5月 | 单词第3轮；阅读精读继续（每周3篇）；总结阅读题型 | 线性代数：行列式、矩阵基础 |
| 第6月 | 单词维持；阅读真题做完近5年；翻译题型入门 | 线代：向量、方程组；概率论基础概念 |
| 第7月 | 单词维持；阅读二刷错题；作文素材积累起步 | 概率：随机变量、期望方差基础 |
| 第8月 | 全科基础查漏；绘制英语+数学知识地图 | 三大模块基础题型成套练习、查漏补缺 |

### 周目标模板（在职期通用）
- **周一~周五**：每天单词打卡 + 1节数学视频课+例题 + 45min英语语法/阅读
- **周六**：本周数学错题重做 + 英语阅读精读2篇（增至4-5h）
- **周日**：本周单词全部回顾 + 数学本章思维导图 + 下周计划微调（增至4-5h）

### 日目标模板（在职期工作日）
- [ ] 单词APP打卡（新词+复习，约30min）
- [ ] 数学：看1节课+跟做3-5道例题（60min）
- [ ] 英语：语法1节 或 阅读精读1篇（45min）
- [ ] 睡前：单词快速回顾 + 默写当日数学公式（15min）

## 阶段二：全职冲刺期（2027年1月 → 2027年12月，约11个月）

### 阶段总目标
- 五大模块全部铺开：英语二、数学、逻辑、写作、434专业课、政治
- 真题反复打磨，进入背诵+模考+查漏的良性循环

### 每日时间分配（8-10h，按番茄钟切块）
| 时段 | 时长 | 内容 |
|------|------|------|
| 上午1 | 7:30-9:00 | 英语（单词+阅读/翻译） |
| 上午2 | 9:15-11:30 | 数学（强化做题，黄金脑力时段给数学） |
| 午休 | — | 必须休息，保护下午效率 |
| 下午1 | 13:30-15:30 | 434专业课（读书+名词解释本+背诵） |
| 下午2 | 15:45-17:30 | 逻辑 或 写作（隔天轮换） |
| 晚间 | 19:00-21:00 | 政治（后期）/ 当日错题整理 / 专业英语 |
> 每天留1-2h弹性缓冲，应对状态波动，避免计划僵死。

### 月度目标拆解

| 月份 | 核心任务 |
|------|---------|
| 2027.1-2 | **数学强化**：396数学全范围系统过+大量做题；英语进入阅读强化+翻译专项；**434启动**：主教材通读+建名词解释本 |
| 2027.3 | 数学强化二轮；**逻辑启动**（形式逻辑+论证逻辑基础）；434第2遍精读 |
| 2027.4 | 数学专题突破（薄弱模块）；逻辑刷题；434整理真题高频考点；英语作文模板起步 |
| 2027.5 | **396真题启动**（近5年成套）；434真题分析+答题模板；**专业英语专项**（上经贸50分） |
| 2027.6 | 396真题二刷+错题；434名词解释开始背；英语二真题成套模考 |
| 2027.7 | **写作专项**（396的论说文+论证有效性分析）；434简答论述背诵；英语作文成篇练习 |
| 2027.8 | **政治启动**（强化班+选择题刷题）；全科真题循环；434背诵第1轮 |
| 2027.9 | 关注各校最新大纲/简章；政治刷题；396全真模考；434背诵第2轮 |
| 2027.10 | **决策点：按模考分数定主攻/保底**；全科模考冲刺；政治分析题背诵起步 |
| 2027.11 | 政治分析题集中背诵；434三轮背诵循环；英语作文/翻译保温；模考查漏 |
| 2027.12 | 全科保温+真题再过；政治时政热点；调整作息对齐考试时间；**初试** |

### 周目标模板（全职期通用）
- **周一~周六**：五大模块按日分配表推进，每天结束整理当日错题
- **周日**：本周全科错题重做 + 1次半天模考（轮换科目）+ 下周计划复盘
> 每周至少留半天彻底休息，全职备考最忌无休导致后期崩盘。

### 日目标模板（全职期）
- [ ] 英语：单词复习 + 阅读/翻译练习并精析
- [ ] 数学：1个专题做题 + 错题归因
- [ ] 434：读书/背诵 + 更新名词解释本
- [ ] 逻辑或写作（隔天轮换）：定量练习
- [ ] 晚间：政治（后期）/ 专业英语 / 当日错题清算
- [ ] 睡前：勾选今日完成项，记录1个最大薄弱点

## 关键决策点（贴书桌前）

> **2027年10月模考分数定主攻：**
> - 若 396稳定≥110 且 英语二≥70 → 上经贸为唯一主攻，浙商大保底
> - 若 396长期≤90 或 英语二≤55 → 锁定浙商大保底，稳上岸优先

## 三条铁律

1. **在职期不贪多**：只做英语+数学基础，其余一律不碰，守住连续性。
2. **数学放黄金时段**：脑力最好的上午留给数学，这是跨考最大短板。
3. **434早建名词解释本、晚集中背诵**：框架尽早搭，背诵主战场放考前两个月。
