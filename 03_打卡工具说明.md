# 打卡工具说明（供后续迭代）

> 最后更新：2026-06-24 · 版本 v4.4（资料包在线观看版）

---

## 一、文件说明

| 文件 | 用途 |
|------|------|
| `index.html` | **主应用**（PWA 版），部署到静态服务器后可通过浏览器打开 |
| `manifest.json` | PWA 清单，定义应用名称、图标、全屏模式等 |
| `icon.svg` | 矢量应用图标（"研"字 + 深蓝渐变背景），一份文件适配所有尺寸 |
| `sw.js` | Service Worker，实现离线缓存，首次访问后断网也能用 |
| `打卡工具.html` | 旧版（保留），双击即可本地打开，仍可用但不含新功能 |

> **PWA 特性**：支持 iOS Safari "添加到主屏幕"、Android Chrome 安装提示、离线使用。

## 二、数据结构（v4）

```js
state = {
  // === v1 字段（保持兼容）===
  p: 0, wk: 1, day: 0,
  data: {},                 // 打卡记录

  // === v2 字段 ===
  startDate: "2026-06-01",
  examDate: "2027-12-25",
  customTasks: {},
  deletedDefaults: {},

  // === v3 字段 ===
  notes: {},                // 每日笔记：键=YYYY-MM-DD

  // === v4 字段 ===
  resources: [],            // 资料包
  studySessions: [],        // 学习会话
  aiChats: [],              // AI 对话记录
  version: 4

```

```js
{
  id: "r_1719000000_abc12",
  subject: "math",           // eng|math|logic|write|major|pol|other
  category: "网课",
  type: "video",             // video|pdf|doc|link|image|folder
  title: "张宇396数学基础课",
  url: "https://...",
  provider: "B站",
  panCode: "a1b2",          // v4.4: 网盘提取码（仅网盘链接用，可选）
  linkedTaskTags: ["t-math"],
  stage: null, month: null, week: null,
  progress: { status:"studying", percent:30, lastPositionText:"看到第5节", lastStudiedAt:"...", videoTime:128.5 }, // v4.4: videoTime=视频续播秒数
  note: "",
  createdAt: "...", updatedAt: "..."
}
```

### studySessions 对象结构
```js
{
  id: "r_...",
  resourceId: "r_...",      // 关联资源（可为 null）
  taskKey: null,
  subject: "math",
  title: "张宇396数学基础课",
  startedAt: "...", endedAt: "...",
  minutes: 20,
  result: "completed",      // completed|partial|not_done
  note: ""
}
```

### notes 对象结构

```js
// 键为 YYYY-MM-DD 日期字符串（由 startDate 推算得出）
"2026-06-23": {
  weakSubject: "math",                      // 薄弱科目
  text: "排列组合公式记混了，需要重做例题",   // 笔记正文（≤100字）
  createdAt: "2026-06-23T14:30:00.000Z"     // ISO 时间戳
}
```

### weakSubject 枚举

| 值 | 含义 |
|----|------|
| `eng` | 英语 |
| `math` | 数学 |
| `major` | 专业课（434） |
| `logic` | 逻辑/写作 |
| `write` | 写作 |
| `pol` | 政治 |
| `other` | 其他 |

### 迁移机制

- v1（无 version）→ 自动补 v2 + v3 字段
- v2（version=2）→ 自动补 notes + 升级 version=3
- 旧 notes 如使用 p/w/d 键格式，首次访问时会按日期键重写（兼容过渡期数据）

## 六、关键逻辑函数

### 基础函数
- `keyFor(p, wk, day)` — 阶段+周+天 → 存储键 `"p0w1d0"`
- `dateFor(p, wk, day)` — 阶段+周+天 → `YYYY-MM-DD` 绝对日期（需设置 startDate）
- `todayStr()` — 返回今天的 `YYYY-MM-DD`
- `monthIdxFor(p, wk)` — 阶段+周 → 第几个月（每月4周）
- `tasksFor(p, dayIdx)` — 取当天任务模板
- `effectiveTasks(p, wk, day)` — 取实际任务列表（自定义 > 默认，过滤隐藏项）
- `focusText(t, m)` — 计算聚焦提示文字
- `computeStreak()` — 连续打卡天数
- `computeTotal()` — 累计完成项数

### 笔记闭环
- `getNote(dateKey)` — 按 YYYY-MM-DD 读取笔记
- `saveNote(dateKey, weakSubject, text)` — 保存笔记
- `renderNotesHistory()` — 渲染最近5条笔记回看
- `renderYesterdayReminder()` — 渲染昨日薄弱点提醒（含3天连续同科警告）

### 分析与可视化
- `getTagStats(daysBack)` — 统计最近N天各 tag 完成率
- `renderBiasWarning(tasks, done)` — 偏科预警（阈值见下方规则）
- `heatmapColor(cnt, total)` — 热力图5色映射
- `renderHeatmap()` — 渲染4周热力图

### 资料包应用内查看器（v4.4）
- `detectPlayMode(r)` — 按链接判定播放方式：`pan`(网盘) / `bili`(B站) / `hls`(.m3u8) / `video`(.mp4等直链) / `pdf` / `office`(Word/PPT) / `image` / `extvideo`(非直链视频) / `link` / `none`
- `openViewer(r, mode)` — 打开应用内查看器弹窗，按 mode 渲染 `<video>` / iframe / Office Online / B站播放器
- `closeViewer()` — 关闭并保存进度，弹出学习会话确认
- `_saveViewerProgress()` — 把视频 `currentTime` 写入 `progress.videoTime`，并据时长算 `percent`（≥95% 标记 done）
- `loadHls(cb)` — 仅在遇到 `.m3u8` 时懒加载 hls.js（CDN），不影响离线
- `copyText(t)` — 复制提取码到剪贴板（网盘链接打开时自动调用）
- `escAttr(s)` / `fmtTime(s)` — iframe 属性转义 / 秒数转 `分:秒`

**路由规则**：`pan`/`link`/`extvideo` → 跳转新窗口（网盘自动复制提取码）；其余可内嵌类型 → 应用内查看器。视频支持记忆进度、重开续播。卡片显示 `▶ 在线播放` 或 `↗ 跳转` 角标。

> **依赖直链**：PDF/Office 内嵌、视频播放都需要「可公开访问的直链」。对象存储私有桶用签名 URL（有有效期）；百度网盘等不提供直链，只能跳转。

### 导入导出
- `exportData()` — 导出 JSON（含 notes）
- `importData()` — 导入 JSON（含格式校验）
- `migrate(loaded)` — 自动数据迁移（`panCode`/`videoTime` 为可选字段，旧数据无需迁移）

## 七、偏科预警规则

### 冷启动保护
- 累计有打卡数据的天数 < 3 → 不显示预警，改为引导文案：`📊 完成 3 天后开始生成偏科趋势`
- 最近 7 天内某科任务总数 < 2 → 该科不参与预警

### 统计范围
只统计考试科目：`t-eng`（英语）、`t-math`（数学）、`t-major`（专业课）、`t-logic`（逻辑/写作）、`t-write`（写作）、`t-pol`（政治）。
**不统计** `t-rev`（回顾/休息）。

### 阈值
| 条件 | 级别 | 视觉 |
|------|------|------|
| 该科最近7天总任务 < 2 | 不参与预警 | — |
| 完成率 < 25% | 红色预警 🔴 | 红色 replan 卡片 |
| 完成率 < 40% | 黄色预警 🟡 | 黄色 replan 卡片 |
| 连续两周 < 40%（仅第2周起） | 连续两周偏低标签 | 红色标签附加 |

### 预警数量限制
- 首页最多展示 **2 条**预警
- 优先级：红色（<25%）> 连续两周预警 > 黄色（<40%）
- 超出 2 条时显示：`还有 X 项提醒未展开`

### 预警文案
每条预警包含**动作建议**，每科有独立的建议文案。

## 八、热力图颜色规则

| 颜色 | 含义 | CSS 类 |
|------|------|--------|
| 灰色 | 当天无任务（无记录） | `hm-gray` |
| 红色 | 有任务但完成率 0% | `hm-red` |
| 黄色 | 完成率 1%–59% | `hm-yellow` |
| 绿色 | 完成率 60%–89% | `hm-green` |
| 深绿色 | 完成率 90%–100% | `hm-darkgreen` |
| 蓝色边框 | 今天（叠加在颜色之上） | `hm-today` |

无图表库依赖，纯 CSS `background` 实现。暗色模式下自动切换深色调配色。

## 九、首页布局（移动端优先）

```
1. 倒计时卡片（距初试天数）
2. 阶段切换
3. 周/月导航 + 今天按钮 + 起始日期设置
4. 今日任务卡片
   ├── 任务列表（含打卡复选框）
   ├── 昨日薄弱点提醒（自动，无记录则隐藏）
   ├── 偏科预警（自动，无预警则隐藏）
   └── 每日笔记（薄弱科目选择 + 文本输入 + 保存状态）
5. 统计卡片
   ├── 本周完成率 / 连续天数 / 累计完成
   ├── 近4周热力图
   └── 最近笔记回看
6. 数据导入导出
```
