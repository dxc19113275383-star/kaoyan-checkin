## 一、文件说明

## 二、数据结构（v4）

```js
state = {
  // === v1 字段（保持兼容）===
  p: 0, wk: 1, day: 0,
  data: {},                 // 打卡记录

  // === v2 字段 ===
  startDate: "2026-06-01",
  examDate: "2027-12-25",
  customTasks: {},
  deletedDefaults: {},

  // === v3 字段 ===
  notes: {},                // 每日笔记：键=YYYY-MM-DD

  // === v4 字段 ===
  resources: [],            // 资料包
  studySessions: [],        // 学习会话
  aiChats: [],              // AI 对话记录

  // === v5/v6 字段 ===
  reminderSettings: {...},  // 学习提醒
  reminderLog: {},          // 提醒日志
  notificationSettings: {...}, // Web Push 推送设置

  // === v7 字段：学习中心 ===
  learn: {
    vocab: {                      // 背单词（百词斩底层逻辑）
      active: "kaoyan",           // 当前词库 id
      settings: { dailyNew: 20 }, // 每日新词配额
      progress: {                 // 键=`${libId}:${word}`
        "kaoyan:abandon": { stage:3, due:"2026-06-28", wrong:0, seenAt:"..." }
      },
      wrongBook: ["kaoyan:abandon"], // 错词本（key 列表）
      daily: { date:"2026-06-24", newDone:5, reviewDone:12 }, // 当日进度（跨天重置）
      userLibs: [ { id:"user_xxx", name:"我的专业英语", count:120 } ] // 导入词库索引
    },
    reading: {                    // 阅读（薄荷阅读式精读）
      progress: { "comparative-advantage": { read:true, readAt:"...", saved:3 } },
      userPassages: [ { id:"u_xxx", title:"我的真题阅读" } ] // 导入文章索引
    },
    math:    {                    // 数学（396 分类练习 + 错题本）
      active: "calc",             // 当前题集 id
      progress: {                 // 键=`${setId}:${qid}`
        "calc:c1": { stage:2, correct:true, wrong:0, seen:3, lastAt:"..." }
      },
      wrongBook: ["calc:c3"],     // 错题本（key 列表，连续答对达 MASTER 后移除）
      daily: { date:"2026-06-25", done:8, correct:6 }, // 当日进度（跨天重置）
      userSets: [ { id:"user_xxx", name:"我的真题·微积分", count:30 } ] // 导入题集索引
    },
    exams:   { progress:{} },     // 真题题库
    mock:    { history:[] }       // AI模拟考记录
  },
  version: 7
}
```

> **词库内容存储**：bundled 词库放仓库 `data/words/*.json`（注册表 `data/words/index.json`），按需 fetch + SW 缓存离线；用户导入词库存 `localStorage['vocab_lib_<id>']`（不进 state，避免膨胀）。`state.learn.vocab` 只存进度/设置，并入导出备份。
>
> **词库 JSON 格式**：`{ id, name, desc, words:[ {w, ph, def, ex, exCn} ] }`，导入时也接受裸数组。`w`(单词)必填，其余可选。

> **题集 JSON 格式**：`{ id, name, cat, questions:[ {id, cat, level, stem, options:[...], answer, explain} ] }`，导入时也接受裸数组。`stem`(题干)/`options`(选项数组)/`answer`(正确项下标,从0起)必填，`explain`(步骤讲解)/`cat`/`level` 可选。bundled 题集放 `data/math/*.json`（注册表 `data/math/index.json`）；用户导入题集存 `localStorage['math_set_<id>']`（不进 state）。

> **存储原则**：内容（词库/题库等只读大文件）走仓库 `/data/` 下静态 JSON，按需 fetch + SW 缓存离线；`state.learn` 只存**进度/设置**（体积小），并入现有导出/导入备份。`defaultLearn()` 提供默认结构，`migrate()` 中 v6→v7 自动补齐、且做了字段缺失防御。

## 六、关键逻辑函数

### 基础函数
- `keyFor(p, wk, day)` — 阶段+周+天 → 存储键 `"p0w1d0"`
- `dateFor(p, wk, day)` — 阶段+周+天 → `YYYY-MM-DD` 绝对日期（需设置 startDate）
- `todayStr()` — 返回今天的 `YYYY-MM-DD`
- `monthIdxFor(p, wk)` — 阶段+周 → 第几个月（每月4周）
- `tasksFor(p, dayIdx)` — 取当天任务模板
- `effectiveTasks(p, wk, day)` — 取实际任务列表（自定义 > 默认，过滤隐藏项）
- `focusText(t, m)` — 计算聚焦提示文字
- `computeStreak()` — 连续打卡天数
- `computeTotal()` — 累计完成项数

### 笔记闭环
- `getNote(dateKey)` — 按 YYYY-MM-DD 读取笔记
- `saveNote(dateKey, weakSubject, text)` — 保存笔记
- `renderNotesHistory()` — 渲染最近5条笔记回看
- `renderYesterdayReminder()` — 渲染昨日薄弱点提醒（含3天连续同科警告）

### 分析与可视化
- `getTagStats(daysBack)` — 统计最近N天各 tag 完成率
- `renderBiasWarning(tasks, done)` — 偏科预警（阈值见下方规则）
- `heatmapColor(cnt, total)` — 热力图5色映射
- `renderHeatmap()` — 渲染4周热力图

### 阅读引擎（v4.7 · 薄荷阅读底层逻辑）
**核心**：精读文章 + 点词查义 + 一键加入背单词（与背单词引擎打通）+ 已读/生词进度。
- `loadReadingList()` / `loadPassage(p)` — 读注册表 `data/reading/index.json` 与文章 JSON；用户导入文章存 `localStorage['reading_<id>']`
- `openReading()` / `renderReadingHome()` — 文章列表 + 已读数 + 导入入口
- `openPassage(id)` — 加载文章，构建查词表 `_readingDict`（本文 glossary + 当前背单词库合并）
- `rdTokenize(text)` — 把段落里每个英文单词包成可点击 `<span class="rd-word">`
- `readingTap(el)` — 点词弹层：释义（查 `_readingDict`，含简单去屈折回退）+ 🔊TTS + 「加入背单词」
- `addWordToReadingLib(entry)` — **关键打通**：把生词写入 `localStorage['vocab_lib_reading']` 并注册为背单词 userLib「阅读生词本」，直接进入 SRS 复习
- `markPassageRead(id)` — 标记已读（可取消）
- **文章 JSON 格式**：`{ id, title, source, level, intro, paragraphs:[...], glossary:[{w,ph,def,ex,exCn}] }`，glossary 可选
- **内容**：附 2 篇原创短文（国际贸易/跨境电商，贴合 MIB，规避版权）；真题/外刊用「导入文章」粘 JSON 加入
- **已知取舍**：生词词条存 localStorage（不进 state/导出备份），换设备后 SRS 进度键会失去对应词条（无害）；与背单词导入词库同一架构

### 数学引擎（v4.8 · 396 分类练习）
**核心**：按题集（微积分/线代/概率）做单选题 + 即时判分 + 内置步骤讲解 + 错题本重做 + AI 详解（仅辅助）。
- `MATH_MASTER = 3`（同题连续答对累计达此 stage = 已掌握，自动移出错题本）
- `ensureMath()` — 取/补 `state.learn.math` 字段；`daily` 跨天自动重置
- `loadMathSets()` — 读注册表 `data/math/index.json` + 合并用户导入题集
- `loadMathQuestions(setId)` — 取题表：用户导入(localStorage)优先，否则 fetch `data/math/<file>`，缓存于 `_mathCache`
- `openMath()` / `renderMathHome()` — 题集选择器 + 题目数/正确率/已掌握三数 + 进度条 + 开始练习/错题本/导入
- `buildMathQueue(qs, mode)` — `practice`=全题集，`wrong`=错题本；洗牌
- `renderMathCard()` — 分类标签 + 题干(`.math-stem`) + A/B/C/D 选项（复用 `.vocab-opt`）
- `mathPick()` / `gradeMath()` — 判分：对→`stage+1`、错→`stage=0`+入错词本；记当日 done/correct
- `showMathFeedback()` — 正误 + 正确答案 + 内置 `.math-explain` 步骤讲解 + 「🤖 AI 详解」「下一题」
- `mathAskAI()` — **AI 辅助打通**：把题干+选项+正确项填入 AI 助手输入框、切到「错题讲解」模式、跳转 AI Tab，由用户发送（AI 仅辅助，注意核对）
- `mathOpenImport()` / `mathImpSave` — 导入题集（粘贴 JSON）
- **内容**：附 3 个种子题集 `data/math/{calc,linear,prob}.json`（微积分/线性代数/概率统计各 8 题，含步骤讲解）；逻辑题集注册为「待导入」占位。真题用「导入题集」粘 JSON 加入。
- **已知取舍**：题干用纯文本/Unicode 写数学符号（无 MathJax 依赖，保证离线零依赖）；复杂公式以可读文字描述。AI 详解走在线代理，离线时仅内置 explain 可用。

### 背单词引擎（v4.6 · 百词斩底层逻辑）
**核心机制**：斩词识别（看单词选释义 4 选 1）+ 艾宾浩斯间隔复习 + 每日新词配额 + 熟词跳过 + 错词本。
- `VOCAB_INTERVALS = [0,1,2,4,7,15,30,90,180]`（天）；`VOCAB_MASTER_STAGE = 7`（达到=已掌握）
- `ensureVocab()` — 取/补 `state.learn.vocab` 字段；`daily` 跨天自动重置
- `loadVocabLibs()` — 读注册表 `data/words/index.json` + 合并用户导入词库
- `loadVocabWords(libId)` — 取词表：用户导入(localStorage)优先，否则 fetch `data/words/<file>`，缓存于 `_vocabCache`
- `openVocab()` / `renderVocabHome()` — 词库选择器 + 今日新词/待复习/已掌握三数 + 进度条 + 开始/错词本/每日配额
- `buildVocabQueue(words, mode)` — `today`=待复习+新词(配额内)，`wrong`=错词本；洗牌
- `renderVocabCard()` — 单词大字 + 音标 + 🔊TTS(`vSpeak`，Web Speech API) + 4 释义选项 + 「我认识」
- `vocabPick()` / `gradeVocab()` — 判分：对→`stage+1`、错→`stage-2`+入错词本；按 `stage` 取间隔算 `due`；记当日计数
- `vocabKnown()` — 熟词一键标记掌握；`vocabImpSave` — 导入词库（粘贴 JSON）
- **图省略说明**：百词斩的"配图"是逐词版权资产，无法离线/合规凑齐，故用文字卡替代，但记忆算法与交互循环一比一还原。
- **内容**：附种子库 `data/words/kaoyan.json`（考研核心 50 词，含大量商务/经济词）；四级/六级/专业英语注册为"待导入"占位。

### 学习中心框架（v4.5）
底部 `资料包` Tab 改名为 `学习`（仍 3 Tab）。「学习」是一个**两层结构**：hub 卡片网格 + 子页内路由（不走底部导航的 showPage，而是 tab 1 内部切换）。
- `LEARN_MODULES[]` — 模块配置数组（key/icon/title/sub/ready/todo），同时驱动 hub 卡片与占位页文案
- `renderLearnHub()` — 按 LEARN_MODULES 渲染卡片网格；`ready:true` 显示「可用」、其余「敬请期待」
- `showLearnView(name)` — tab 1 内部路由：`hub`(中心) / `resources`(资料网课，复用查看器) / 其余 key → 通用占位页 `pageLearnTodo`
- 页面 id：`pageLearn`(hub) / `pageLearnTodo`(占位) / `pageResources`(资料网课)
- `showPage(1)` 进入即 `showLearnView('hub')`；各子页有「← 返回学习」回 hub
- **当前状态**：框架/导航/数据模型已就位；`资料/网课` 已可用，其余 6 个引擎为占位，逐个填充不动框架

### 资料包应用内查看器（v4.4）
- `detectPlayMode(r)` — 按链接判定播放方式：`pan`(网盘) / `bili`(B站) / `hls`(.m3u8) / `video`(.mp4等直链) / `pdf` / `office`(Word/PPT) / `image` / `extvideo`(非直链视频) / `link` / `none`
- `openViewer(r, mode)` — 打开应用内查看器弹窗，按 mode 渲染 `<video>` / iframe / Office Online / B站播放器
- `closeViewer()` — 关闭并保存进度，弹出学习会话确认
- `_saveViewerProgress()` — 把视频 `currentTime` 写入 `progress.videoTime`，并据时长算 `percent`（≥95% 标记 done）
- `loadHls(cb)` — 仅在遇到 `.m3u8` 时懒加载 hls.js（CDN），不影响离线
- `copyText(t)` — 复制提取码到剪贴板（网盘链接打开时自动调用）
- `escAttr(s)` / `fmtTime(s)` — iframe 属性转义 / 秒数转 `分:秒`

**路由规则**：`pan`/`link`/`extvideo` → 跳转新窗口（网盘自动复制提取码）；其余可内嵌类型 → 应用内查看器。视频支持记忆进度、重开续播。卡片显示 `▶ 在线播放` 或 `↗ 跳转` 角标。

> **依赖直链**：PDF/Office 内嵌、视频播放都需要「可公开访问的直链」。对象存储私有桶用签名 URL（有有效期）；百度网盘等不提供直链，只能跳转。

### 导入导出
- `exportData()` — 导出 JSON（含 notes）
- `importData()` — 导入 JSON（含格式校验）
- `migrate(loaded)` — 自动数据迁移（`panCode`/`videoTime` 为可选字段，旧数据无需迁移）

## 七、偏科预警规则

### 冷启动保护
- 累计有打卡数据的天数 < 3 → 不显示预警，改为引导文案：`📊 完成 3 天后开始生成偏科趋势`
- 最近 7 天内某科任务总数 < 2 → 该科不参与预警

### 统计范围
只统计考试科目：`t-eng`（英语）、`t-math`（数学）、`t-major`（专业课）、`t-logic`（逻辑/写作）、`t-write`（写作）、`t-pol`（政治）。
**不统计** `t-rev`（回顾/休息）。

### 阈值
| 条件 | 级别 | 视觉 |
|------|------|------|
| 该科最近7天总任务 < 2 | 不参与预警 | — |
| 完成率 < 25% | 红色预警 🔴 | 红色 replan 卡片 |
| 完成率 < 40% | 黄色预警 🟡 | 黄色 replan 卡片 |
| 连续两周 < 40%（仅第2周起） | 连续两周偏低标签 | 红色标签附加 |

### 预警数量限制
- 首页最多展示 **2 条**预警
- 优先级：红色（<25%）> 连续两周预警 > 黄色（<40%）
- 超出 2 条时显示：`还有 X 项提醒未展开`

### 预警文案
每条预警包含**动作建议**，每科有独立的建议文案。

## 十二·补、设计系统（v4.9 起 · 重构中）

> 目标：从"工具 demo"升级为"高级、有质感、有陪伴感的考研备战系统"。遵循 minimalist-ui 为主、high-end-visual-design 克制取用。分阶段（P1 底座 → P2 首页驾驶舱 → P3 AI 教练 → P4 资料中心 → P5 打卡成就 + 移动端/PWA 收尾），每阶段独立可用、不破坏功能、单独提交。

### 设计令牌（CSS 变量，定义于 `:root`，含暗色推导）
- **品牌主色**：深墨绿 `--brand #1f4d3f`（暗色 `#2f7d62`）。**关键手法**：legacy `--color-text-info` 一并重指为墨绿——倒计时/主按钮/进度条/选中态/复选框/底栏/链接全部一处改全局。
- **强调色**：暖橙 `--ember #c2570f`（暗色 `#e08a4f`）+ `--ember-soft`，**稀缺使用**：连续天数、冲刺必看、关键鼓励。
- **中性**：暖白基底 `--color-background-tertiary #faf9f5`、卡面 `#fff`、次级 `#f1efe8`；统一发丝边框 `--hair`/`--hair-strong`。
- **阴影**：`--shadow-ambient`（卡片浮起）/`--shadow-key`（hero/弹层），带品牌色相、极低不透明，替代纯边框扁平感。
- **半径**：`--border-radius-sm 8 / md 10 / lg 14 / --radius-xl 20 / --radius-pill 999`。
- **缓动**：`--ease cubic-bezier(.16,1,.3,1)`、`--ease-out`。
- **字体**：`--font-sans`（系统栈）+ `--font-serif`（系统 serif 备选）。

### 排版
字重层级（400/500/600）、大标题负字距、**数据全量 `tabular-nums`**、`100dvh`、`-webkit-font-smoothing`。

### 组件库（P1 新增，供后续阶段复用）
`.eyebrow`（微标签）· `.btn-primary/.btn-ghost` · `.btn-cta`（含 `.cta-ico` button-in-button 尾部圆形图标）· `.chip`+`.chip-row`（快捷问题/筛选，横滑）· `.badge` ×5 状态（`.b-read/.b-new/.b-rec/.b-freq/.b-sprint`）· `.ring-wrap`（SVG 进度环）· `.empty-state` · `.skeleton`（骨架屏）· `.reveal`（滚动入场，配 IntersectionObserver）。

### 交互态
`:focus-visible` 焦点环、`active:scale` 按压物理、`@media(hover:hover)` 悬浮抬升、`@media(prefers-reduced-motion)` 守卫。

### 关键设计取舍
- **字体刻意保留系统栈**：两个设计 skill 都建议换字符字体，但本项目是**离线优先 PWA + 中文为主 + 历史"去 CDN"原则**。加载 Web 字体会破坏离线、且对中文字形几乎无效；改为在系统栈内用字重/字距/tabular-nums/serif 备选拉开质感。如需 Latin 展示字体，后续可自托管 woff2 + SW 预缓存。
- **拒绝**：玻璃拟态滥用、大旋转、营销级 `py-40` 动效、`rounded-full` 大容器。**取用**：自定义缓动、按压物理、嵌套层级、tinted 阴影、eyebrow、button-in-button。
- **emoji**：结构 UI 逐步替换为线性 SVG；人声文案（鼓励/连续天数）保留极少量以保陪伴感。
